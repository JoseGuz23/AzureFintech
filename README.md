# Dashboard Financiero Fintech

Proyecto de tesis: Diseño e Implementación de una Arquitectura de Servicios en la Nube para Entornos Fintech en Microsoft Azure

## Características

- Autenticación con Microsoft Entra ID
- Dashboard con análisis financiero en tiempo real
- Gestión de transacciones CRUD
- Arquitectura serverless en Azure

## Tecnologías

- React 18
- Azure Static Web Apps
- Azure Functions
- Cosmos DB
- Microsoft Entra ID
